# RetrofitComposeSample (Proyecto Android - ejemplo)

## Cómo abrir en Android Studio
1. Descarga y descomprime el ZIP.
2. En Android Studio: File → Open... → selecciona la carpeta donde descomprimiste (la carpeta que contiene `settings.gradle.kts`).
3. Espera a que Gradle sincronice.
4. Ejecuta la app en un emulador o dispositivo.
